## 📋 Overview

Riverpod Gen Shortcuts is a comprehensive tool for generating Riverpod code efficiently. This project provides:

- 🎯 **Code Generation** - Automatically generate Riverpod provider boilerplate
- 📚 **Documentation** - Complete guides and examples (Beta)
- 🔌 **Live Plugin** - Ready-to-use plugin for your projects
- 🚀 **Quick Start** - Get up and running in minutes

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- npm, yarn, or pnpm

### Installation & Development

1. **Clone the repository**
   ```bash
   git clone https://github.com/vinit105/riverpod-gen-shortcuts.git
   cd riverpod-gen-shortcuts
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Run development server**
   ```bash
   npm run dev
   ```

4. **Open in browser**
   Navigate to [http://localhost:3000](http://localhost:3000)

### Build & Production

```bash
# Build for production
npm run build

# Start production server
npm start
```
